# Ubuntu Post-install Scripts
My personal post-install script for Ubuntu.

(Tested on Ubuntu 22.04)

# How to use

## Gnome Setup

```bash
./gnome-setup-1.sh
# log out and back in
./gnome-setup-2.sh
```

## Remove snapd
```bash
./remove-snapd
```

## ZSH setup
```bash
./zsh-setup
# log out and back in
```
NOTE: This script requires the password twice
